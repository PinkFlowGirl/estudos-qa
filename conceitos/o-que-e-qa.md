# O que é QA (Quality Assurance)
QA significa Garantia da Qualidade e tem como objetivo assegurar que o software funcione corretamente.
O profissional de QA atua para prevenir erros e garantir que o produto tenha qualidade antes de chegar ao usuário final.
QA não é apenas encontrar bugs, mas também melhorar processos e a experiência do usuário.
O trabalho de QA começa desde o entendimento dos requisitos até a entrega do software.
## Qual a importância do QA
QA é importante porque ajuda a evitar falhas, retrabalho e problemas para os usuários.
Com testes bem feitos, o software se torna mais confiável e seguro.
## O que um QA faz
Analisa requisitos para entender como o sistema deve funcionar.
Cria casos de teste para validar funcionalidades.
Executa testes manuais e registra bugs encontrados.
Acompanha correções e garante que o erro foi resolvido.
